第一节：名词
	1.名词的数：规则
		1.名词直接+s desks，books
		2.字母s，x，ch，sh结尾的+es watches， messes
		3.字母f，fe结尾，需将其改为v，再加es knives，shelves
		4.“辅音字母+y”，需要将y改为i，再加es cookies
		5.“辅音字母+o”，直接加s heros，tomatos
		6.如果是外来词，则加es：redioes，pianoes， photoes， studioes	
		7.不规则变化： man-men； foot-feet； child-children
		8.单复数一样：sheep， deer
		9.只有复数的名词：works（商品）， glasses， goods， chopsticks
		10.用单数表示复数：police；people
	2.不可数名词
		1.物质名词：如air（空气），bread（面包），food（食物），ice（冰），soup（汤）等
		2.抽象名词：如work等
	3.名词所有格：规则
		1.有生命名词后加's表示从属：father's coat （爸爸的外套）
		2.若名词以复数s结尾，直接加“ ' ”:Teachers' Day
		3.一个" 's "为单数，两个“ 's ”为复数。
			Tom and Jim's room + is ; Tom's and Jim's room +are
		4.表示时间，距离，国家，城市，团体，机构等名词后：ten minutes' walk(十分钟的路程)
		5.无生命的名词，一般用of表示所有格：music of the film（电影的音乐）
		6.双重所有格的表示法：a friend of father's(我爸爸的朋友)

第二节：形容词
	1.形容词与副词的区别
		1.adj，adv
	2.比较级与最高级
		1.比较级的表现形式分为两种：若为一到两个音节，则直接加er，strong-stronger。若为三个或以上音节，则需要加more。
		2.最高级的特点之一是，需要在形容词后加est，并且要用定冠词the，China is the greatest country in the world.
	3.不定代词+形容词
		定义：没有明确指哪一个，如：everyone，something
	4.系动词+形容词
		系动词包括 be动词和感官动词 如听起来，feel等
	5.多个形容词的顺序
			绘	大	行	星，	燕	国	制	图
		 	|	 |	 |	 |	 |	 |	 |	 |
			描绘	大小	形状	新旧	颜色	国家	材质	用途
			| 	 |	 |	 | 	 |	 |	 |	 |
		 	beautiful	 big	 圆的	 new	 red	 Chinese	 木头	 ……
	6.as...as...表示倍数
		1.as + adj./adv.原级 + as
		2.as + adj. + n. + as
		3.twice + as + adj.原级 + as 如果有倍数要先写倍数
		4.twice + as + adj + n + as 
	7.the more..., the more...,; more and more
	   越...越...			越来越...
		注意：不一定是more，也可能是其他比较级

第三节：副词
	1.副词位置关系
		1.动词加副词 v. + adv.
		2.副词加形容词 adv. + adj.
	2.形容词如何转化为副词
		1.以le结尾去e变y，possible==》possibly；
		2.在形容词后直接加ly，usual==》usually；
		3.在以辅音字母+e结尾直接加ly，wide==》widely；
		4.以元音字母+e结尾，去e加ly，true==》truly；
		5.以辅音字母+y结尾变y为i+ly，happy==》happily；	
	3.much too / much more + 形容词或副词
		1.much too 太......   
		2.much more 过于......

第四节：冠词
	1.不定冠词a/an
		1.a用在辅音音素之前；an用在元音音素之前；如an hour
		2.泛指某一个或者某一类
		3.用于某些固定搭配中：in a minute；for a minute
				in表示范围		for表示持续
	2.定冠词the
		1.谈话双方都知道的人，事，物	the gate of the school
		2.世界上独一无二的事物	the earth 地球
		3.序数词，最高级，方位等之前	the first ；the north
		4.报刊，杂志之前
		5.乐器前	the piano 
		6.姓氏复数之前，表示一家人	the wangs + 复数
		7.固定搭配
	3.零冠词 什么都不加
		1.专有名词，不可数名词 china water guangzhou
		2.季节，月份，星期，节日前 April  On Friday
		3.三餐，球类运动前 Labor's Day 劳动节 have lunch 吃午餐 play football
		4官衔，职位前 chairman主席 monitor 班长/监视器
		5.固定搭配

第五节：代词
	1.theirs=their + n.等，yours=your+n.，mine=my+n.
	2.不定代词：没有明确指哪一个 不定代词+adj.
		如：all，none，both（两者都），any（任何一个），every（每一个）
			         +复数
	3.both	either		neither	none	some	all
	  两者都  两者中的一个              两者都不    三者都不                   >=三者都
	4.other	others	the other		the others
		1.other：其他的
		2.others=other+people/thing
		3.the other ：两者中的另一个
		4.the others ：剩下的
		5.another：>=三者另一个
		6.in one way or another 无论如何，不管怎样
	5.it做形式主语或形式宾语
		真正主语在：
			1.to do : It is important to finish on time(准时完成是重要的) 
			2.doing：It is no use crying for fight(为打架而哭泣是没有用的)
			3.that从句：It is important that we should go to school(我们去上学是重要的)
第六节：介词 
	介词+名词  介词 + ving 介词+人称代词宾格
	1.at(点，时间),on(线面，天),in(范围，年)
	2.to(去，普遍的对象),from(来),for(持续，特定为谁，特定的对象)
		例子：open to ：对谁开放
		         open for ：为谁开放
	3.by(通过)，over(跨)，against(反)
	4.between(两者之间)，among(三者之间)
	5.across(表面经过)，through(内部穿过)
	6.beside(在...旁边),besides(除了...之外),except(同类，排除在外，除了...)，except for(非同类)，but(不定代词+but：nothing but仅仅只是，anything but绝不是)，apart from(只是)
第七节：连词
	1.and(),but(转折),or(用于否定句，并列，否则)前后结构一致
	2.either...or...(或者...或者...), neither... nor(既不...也不...)， both...and...(两者都)
	3.although/though（包含虽然但是）
	4.in spite of = despite(尽管)
	5.in order to;so as to（目的，为了...）; in order that; so that （+从句）
第八节：数词
	1.基数词：
		365--three hundred and sixty-five
		505--five hundred and five
		1005--one thousand and five
	2.基数词变序数词：
		1.一般加th
		2.第一二三是first，second，third
		3.八去t，九去e；ve用f替（twelve-twelfth）
		4.ty变ti，再加e（twenty-twentieth）
		5.几十几把个位变成序数词（twenty-one==》twenty-first）
	3.数词的应用：
		时刻表达：
			整点=基数词+o`clock（it`s eight o`clock now.）
		几点几分：
			先小时后分（three twenty-five 三点二十五分）
			先分后小时（<30分钟==》past: twenty-five past three 三点二十五）（>30分钟==》to: five to three 差五分钟三点，两点五十五）
			30分钟=falf	15分钟= a quarter
	4.日期表达：
		1949年10月1日（October1st，nineteen forty-nine 先月日，再年）
		a+序数词=又一；再一
		分数： 分子基数，分母序数（分子超过1时分母加s）（三分之二，two thirds）(含分数的短语作主语，由分数后的词决定谓语)
第九节：动词的时态
	1.一般态：
		1.一般现在时：am is are表示频率 
		2.一般过去时：was were过去发生的事实。
		3.一般将来时：will do
		4.一般过去将来时：would （will的过去式）
	2.进行态：be doing
		1.现在进行时：is /am/are doing
		2.过去进行时：was/ware doing
		3.将来进行时：will be doing
		4.过去将来进行时：would be doing
	3.完成态：
		1.现在完成时：过去发生的事对现在的影响has/have+done
		2.过去完成时：过去的过去发生的事对过去的影响had done	
		3.将来完成时: by+将来的时间（whill have/has done ） ==》将来完成；by+过去的时间==》过去完成
		4.过去将来完成时：would have done
	4.完成进行态：has/have been doing  (has/have done + be doing(现在完成+现在进行))
		1.现在完成进行时：have/has been doing
		2.过去完成进行时：had been doing
		3.将来完成进行时：will have been doing
		4.过去将来完成进行时：would have been doing
	5.情态动词：情感态度的动词
		如：must，can/could, would.will, should=ought + to, need, shall
		考点：
			现在：情+do（动词原型）
			过去：情+have done 
				1.must + have done 有证据肯定的猜测。
				2.could have done 本能够，实际没
				3.should have done = ought to have done 本应该，实际没
				4.can`t have done 不可能
				5.needn`t have done 本没必要，实际有
第十节：动词的语态（be done）被动
	1.一般态被动
		1.一般现在时被动：be done； am done； are done
		2.一般过去时被动：was done； were done
		3.一般将来时被动：will be done
		4.一般过去将来时被动：would be done
	2.进行态被动
		1.现在进行时被动：am/is/are being done
		2.过去进行时被动：was/were being done
		3.将来进行时被动：will be being done
		4.过去将来进行时被动：would/should be being done
	3.完成态被动
		1.现在完成时被动：has/have been done
		2.过去完成时被动：had been done
		3.将来完成时被动：will have been done
		4.过去将来完成时被动：would have been done
	4.完成进行态被动
		1.现在完成进行时被动：have been being done
		2.过去完成进行时被动：had been being done
		3.将来完成进行时被动：will have been being done
		4.过去将来完成进行时被动：would/should have been being done
第十一节：非谓语
	把一个动词变成不是动词，就是非谓语，可以变成名词，形容词，副词
	1.ing（主动，正在进行），ed（被动，已发生），to do（未做，将做）
	2.动词+ing（只能加ing）{
			enjoy,consider,can`t help,feel,like,risk,suggest,practice
		}
	3.动词+ing（已做）/to do（去做）
		1.stop+doing/to do	停止做sth/停止去做sth
		2.forget+doing/to do  忘记做过sth/忘记去做sth
		3.remember+doing/to do  记得做过sth/记得去做sth
		4.mean+doing/to do  
		5.regret+doing/to do  后悔去做sth/对某事遗憾	
		6.try+doing/to do	试着做sth/努力地去做sth
		7.go on+doing/to do  继续做sth/继续做另外一件事
	4.连词或介词 + 否定（not）+非谓语
	5.with + 名词 + ing主动/ed被动
	6.have/get/find/want sth. Done
	7.一般现在时：ing（主动），ed（被动），to do（未做），to be done （未被做）
	8.一般过去时：主动：优先使用having done（先后），或ing；被动：ed。
	***********
		一个句子中有两个谓语时，把多出来的谓语变成非谓语（名词，形容词或副词），to do, doing(现在分词) ,done（现在分词）。
		分析时态：如果非谓语发生在谓语所表示的动作之前用do的过去分词doing(表示正在进行的)，发生在之后用to do（未发生）（主动语态变名词，形容词，副词规则一样）
		分析语态：主被动。
			变形容词或副词：doing被动done，to do被动 to be done
			变名词：doing被动 being done，to do被动 to be done
第十二节：简单句
	1.主+谓 = 句子
	2.句子种类：陈述句，疑问句（？，倒装v.+n.），祈使句（动词原形开头，go out），感叹句（！，what how开头）
	3.句子成分：
		主语：一个句子叙述的主体
		谓语：主体发出的动作或状态
		宾语：主体发出动作的状态
		定语：修饰名词或代词
		状语：修饰动词，形容词，副词或整个句子
	4.反义疑问句：
		1.前否后肯，前肯后否。
		2.一般过去时：
			动词 + ed ==》 didn`t + 主
			was/were ==》 wasn`t/weren`t + 主
		3.现在完成时:has/have done ==	》 hasn`t/haven`t + 主
		4.过去完成时：had done ==》 hadn`t + 主
		5.used to do（过去常做的事） ==》didn`t + 主/usedn`t + 主
		6.祈使句 ==》 will you
		7.There be ==》 ben`t there
* 从句：一个句子变成名词，形容词，副词（连词+句子）
	1.作名词：主语从句，宾语从句，同位语从句
	2.作形容词是定语从句
	3.作副词是状语从句

第十三节：名词性从句
	*从句中是陈述句语序，一个句子有且只有一个谓语（动词）
	1.主语从句：
		例:（what I do）is right 有两个动词，就是两个句子
		       主语从句
		引导主语从句的连词
			that whether if what who whom whose which when where how why 
			if做形式主语，真正的主语在后边
	2.宾语从句：
		例：We can wear (what we like).
				宾语从句
	3.同位语从句：fact/idea/news/promise/suggestion/belief + that
第十四节：定语从句
	*定语从句（作形容词，修饰名词）（关系词是连词，先行词是修饰的名词）
	根据连词后的句子(从句)是否完整（主谓宾）来选择关系代词还是关系副词
		不完整：用关系代词
			1.缺主语用who，缺宾语用whom，如果需要“...的”用whose
			2.用which情况：前面有介词，前面有逗号，
			3.用that情况：1.不定代词+that； 2.句首the only+that； 3.序数词+that
		完整：用关系副词（根据先行词选择）
			1.先行词是时间，从句用when
			2.先行词数是地点，从句用where
			3.先行词是....的原因，从句用why
	
第十五节：状语从句
	*作副词，状语从句专属连接词+一个句子 = 副词
	连接词：
		1.时间：when(在...时)，as(当...时)，while(在...期间)，as soon as(一...就...),before(在...之前)，after(在...之后)，since(自从...以来)，not...until(直到...才)， until/till(直到...时)
		2.地点：where(哪里)，wherever（无论哪里），anywhere（任何地方），everywhere（到处）
		3.原因：because(因为)，since（自从），as（作为）
		4.目的：so that（以便），in order that（为了）
		5.结果：so...that(所以...那)，such...that(这样...那样),so that(以便...)
		6.条件：if(如果)，unless（除非），whether（是否）
		7.让步：though（虽然），although（虽然，比though更正式）even if（即使），even though（尽管）
		8.比较：as（同级比较），than（不同程度比较）
		9.方式：as（作为），as if（犹如）
